Some Work

Here are a few of my personal projects I am working on right now.

Mona's Escape

Mona's Escape is a small HTML5 + JavaScript game that I worked on for GitHub's GameOff for 2016. I used PIXI.js as my rendering engine and then built from the ground up.

brightScreen

brightScreen is an idea that came to me when a client whose project was finished had some questions about some code. With brightScreen two users can pull up code alongside one another and work together more seemlessly than in simple messaging apps.

eContract

eContract is a simple one page tool for creating simple written agreements with a built-in text editor converting Markdown to HTML. This tool also has a signature pad that allows for mouce, pen, and touch input. All the information can be stored in a database and printed with the simple click of a button.
