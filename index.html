Some Projects

Here are a few of my personal projects I am working on right now.

Mona's Escape

Mona's Escape is a small HTML5 + JavaScript game that was highlighted by GitHub for GitHub's GameOff for 2016. I used PIXI.js as my rendering engine and then built from the ground up.

brightScreen

brightScreen is an idea that came to me when a client whose project was finished had some questions about some code. With brightScreen two users can pull up code alongside one another.

The Ferry Man

The Ferry Man, is a Knockout.JS SPA that pulls from several different APIs to present data about locations in Florance that are impirtant to Dante's epic poem.

Multi-User Blog Engine

This was something I worked on for my Udacity Full Stack Web Developer Nanodegree. In this project I gained experience with very practical techniques of handling a multi-user/social platform in Python.
